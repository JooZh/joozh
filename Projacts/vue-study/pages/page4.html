过渡（动画）

动画的本质：

实际上vue的动画走的是 css3 的 transition,animation

示例:

简单的隐现动画

代码解析：

1. 在html中给动画元素加上 [ transition="动画名称" ] 

<div id="div1" transition="fade"></div>

2. 在css中添加对应的动画样式

/*动画的样式*/
.fade-transition{
  transition: all 1s ease;
}
/*进入时候的动画*/
.fade-enter{
  opacity: 0;
}
/*离开时候的动画*/
.fade-leave{
  opacity:0;
}

ps: 动画样式，进入动画，离开动画 必须存在。

使用动画库:如Animation.css

除了使用 animation 还能使用别的库

示例:

特效的隐现动画

1. 下载并且引入Animation.css

<link rel="stylesheet" href="../assets/animate.css">

2. 在html中给动画元素添加 animation 动画类

<div id="div2" class="animated"></div>

3. 在html中给动画元素设置动画名称(名称随意，要求有意义)

<div id="div2" class="animated" transition="zoom"></div>

4. vue的 transitions 中定义

transitions:{
  zoom:{
    enterClass:'zoomInLeft',
    leaveClass:'zoomOutRight'
  }
}

ps：transitions和methods是同样的vue属性

Js源码：

new Vue({
  el:'body',
  data:{
    bSign1:true,
    bSign2:true,
  },
  methods:{
    toggle1:function () {
      this.bSign1 = !this.bSign1;
    },
    toggle2:function () {
      this.bSign2 = !this.bSign2;
    }
  },
  transitions:{
    zoom:{
      enterClass:'zoomInLeft',
      leaveClass:'zoomOutRight'
    }
  }
});
